주의사항

  1. <label for="">와 input, select의 id값을 맞춰주세요.(퍼블리싱 할때 임의로 입력) ex) <input type="radio" id="kmh_1" /><label for="kmh_1">초등학교</label>
  2. 스크립트 작성시
    1. <script type="text/javascript">
      //code
      </script>
    2. <a href="javascript:FUNCTION"></a> (X) <a href="#" onclick="FUNCTION(); return false;"></a> (O)
    3. <a href="#" onclick="window.open('URL'); return false;"></a> (X) <a href="URL" onclick="window.open(this.href); return false;"></a> (O)
