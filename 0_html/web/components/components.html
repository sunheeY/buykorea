3-3. 달력 라이브러리

  1. ① 사용 라이브러리:flatpickr
  2. ② handleDateChange:함수는 날짜가 선택될 때 실행되는 콜백 함수로, 선택된 날짜를 콘솔에 출력합니다.
  3. ③ Flatpickr 옵션
    • dateFormat : 날짜 표시 형식을 설정합니다. 예를 들어, 'Y-m-d H:i'는 '2023-09-12 15:30'과 같은 형식으로 표시합니다.
    • enableTime : 시간 선택을 활성화하거나 비활성화합니다.
    • time_24hr : 24시간 형식을 사용하도록 설정합니다.
    • minDate : 선택 가능한 최소 날짜를 설정합니다. 여기서 'today'는 오늘 이전 날짜 선택을 비활성화합니다.
    • defaultDate : 초기 날짜를 설정합니다. 'today'는 현재 날짜로 설정합니다.
    • disable : 특정 날짜를 비활성화합니다.
    • locale : 지역화 설정으로 한국어를 사용하도록 설정합니다.
    • onChange : 날짜가 변경될 때 실행되는 콜백 함수를 등록합니다. handleDateChange 함수가 여기서 사용됩니다.
    • disableMobile : 모바일 기기에서 위젯을 비활성화합니다.
    • altInput 및 altFormat : 추가 입력란을 활성화하고 해당 형식을 설정합니다.
    • mode : multiple,range
    • inline: 달력을 기본 보이는 상태로 적용할지 활성화 하거나 비활성화합니다.
