          반려로 상태를 변경하면 소속직원 목록에서 직원은 비노출됩니다.
        • 승인: 기업에 소속된 직원, 셀러 활동이 가능한 직원은 승인 상태로 변경해 주세요.
        • 정지: 기업에 재직중이나, buyKOREA에서 셀러로 활동이 필요 없는 경우 ex) 육아휴직 정지 상태인 직원은 셀러 활동이 어렵습니다.
        • 해지: 기업에 더 이상 셀러 활동을 하지 않는 경우 ex) 퇴사 해지로 상태를 변경하면 소속직원 목록에서 직원은 비노출됩니다.
      • 정지/해지 상태 직원은 셀러 활동이 어렵습니다.
        해당 직원의 담당 상품은 다른 직원에게 상품 이관 후 진행해 주세요.
